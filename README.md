# Blokus
A Blokus game built based on ~~friendship~~ using Java

## Requirements
- [Java SE Development Kit 13](https://jdk.java.net/13/)

## Running
Run through Visual Studio Code's Code Runner, or enter 
```
java Blokus.java
```
in your preferred commandline.
